About CS 5610

Web applications are applications that use various Web technologies as building blocks. In CS 5610, HTML, CSS, JavaScript and AngularJS are used to build the front directly interacting with users. As users interact with the Web application they generate a sequence of HTTP requests directed at a server built on Node.js and Express, a popular Node.js library. The server handles the requests and use Mongoose to store and retrieve a data model from MongoDB, a NoSQL database. The front end, web services, and data access and storage are all hosted and/or configured in a Node.js runtime instance.
